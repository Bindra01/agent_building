# Agentic Physics Tutor

This project implements a multi-step agentic system that:

- Parses physics PDFs
- Detects nuanced reasoning
- Generates detailed derivations
- Performs internal verification
- Optionally performs external web verification using Tavily

## Tech Stack
- OpenAI API
- Tavily Web Search
- Jupyter Notebook

## How to Run
1. Clone repo
2. Create .env file
3. Add API keys
4. Run notebook
